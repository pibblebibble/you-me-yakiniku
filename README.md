# Jon &amp; Viv: We Ball

The friends' wedding RSVP site — Saturday **3 October 2026** at Bandar Puteri Puchong.

Bold blue + chunky doodly aesthetic. Static HTML / CSS / JS, hosts free on GitHub Pages.

## Pages

- `index.html` — Welcome / invitation
- `details.html` — Schedule + locations + dress code
- `rsvp.html` — Embedded Jotform
- `thank-you.html` — Custom confirmation page (Jotform redirects here)

Both on Bandar Puteri Puchong, across from each other:
- **Reception:** Ground Eatery (4pm onward)
- **Dinner:** Ichigo e-Yakiniku

## Dress code

Smart Casual. BBQ friendly.

## Form submissions

Embedded Jotform: `https://www.jotform.com/form/261533465264458`. Responses land in the Jotform dashboard.

## Design tokens

Palette: bold blue (`#224da5`) on white. Type: Permanent Marker (display) + Patrick Hand (accent) + Inter (body). Defined in `css/style.css`.
